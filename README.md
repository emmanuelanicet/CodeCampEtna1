# Introduction 

Défi lancer par notre école, qui consiste à créer un système émargement via nos cartes. 

## QR-code parse les donées 

Installation d'expo : 
```
npm install --global expo-cli
```
Initialiser l'application sous le nom donnée 
```
expo init my-app
```
Ouvrir le dossier 
```
cd my-app
```
Demarrer le projet 
```
expo start
```
